# Electrical Problems Diagnosis

## Common N14 Electrical Issues:

### Check Engine Light:
- See: [Check Engine Codes](check-engine-codes.md)

### Starting Issues:
- Battery and alternator - Check voltage (12.6V rest, 13.5-14.5V running)
- Starter motor - Listen for clicking vs cranking
- See: [No Start Diagnosis](no-start.md)

### Sensor Failures:
- Crankshaft position sensor - P0335
- Camshaft position sensors - P0340/P0345
- Oxygen sensors - P0420/P0430
- See component guides in `../engine/components/sensors-actuators/`

### Ignition System:
- Spark plugs - Replace every 96,000 km
- Ignition coils - Common failure
- See: [Misfire Diagnosis](misfire-diagnosis.md)

**For detailed electrical component information, see:** `../electrical/electrical-components.md`
