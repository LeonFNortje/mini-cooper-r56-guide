# Check Engine Light & OBD-II Codes

**Common N14 Codes:**

For specific code diagnosis, see the relevant component guides:

## Timing Chain Codes:
- **P0016, P0008, P0009** → [Timing Chain Guide](../engine/components/valvetrain-timing/01-timing-chain.md)

## Fuel System Codes:
- **P0087, P0088** → [HPFP Guide](../engine/components/fuel-injection/01-HPFP.md)
- **P0190-P0193** → [HPFP Diagnosis](HPFP-diagnosis.md)

## Misfire Codes:
- **P0300-P0304** → Check spark plugs, ignition coils, carbon buildup
- Carbon buildup: [Carbon Guide](../engine/components/intake-exhaust/04-carbon-buildup-guide.md)
- Valve cover oil in wells: [Valve Cover Guide](../engine/components/cylinder-head/04-valve-cover.md)

## General Diagnosis:
Use an OBD-II scanner to read codes, then refer to specific component guides for detailed diagnosis.
